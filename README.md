# Programación Orientada a Objetos - Reto 1

Este repositorio contiene la solución a los ejercicios propuestos en clase.

## 📌 Ejercicios

### 1. Calculadora básica
Se implementó una función que recibe dos números y un operador (+, -, *, /) y retorna el resultado según la operación.

### 2. Palíndromo
Se creó una función que verifica si una palabra es palíndromo sin usar slicing. Se comparan los caracteres manualmente.

### 3. Números primos
Se desarrolló una función que recibe una lista de enteros y retorna únicamente los números primos.

### 4. Suma consecutiva
Se implementó una función que recorre una lista y calcula la mayor suma entre dos números consecutivos.

### 5. Anagramas
Se creó una función que identifica palabras que contienen los mismos caracteres (anagramas).
